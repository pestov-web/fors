# UI-Kit: Система дизайна Автошкола Форсаж

## Обзор

UI-Kit содержит переиспользуемые компоненты пользовательского интерфейса, которые используются по всему сайту. Каждый компонент разработан согласно дизайну из Figma и следует единым принципам.

## Структура

```
ui-kit/
├── buttons.css      # Система кнопок
├── forms.css        # Формы и поля ввода (планируется)
├── cards.css        # Карточки (планируется)
└── README.md        # Эта документация
```

## Компоненты

### 1. Кнопки (buttons.css)

#### Типы кнопок

**Primary (Красные)** - Основные CTA кнопки
- Цвет: `#f2404f` → `#ff6670` (hover)
- Использование: главные призывы к действию ("Записаться на курс")

**Secondary (Желтые)** - Альтернативные CTA
- Цвет: `#f2d502` → `#ffd700` (hover)
- Использование: вторичные действия, формы

**Outline** - Прозрачные с обводкой
- Использование: дополнительные действия, навигация

**Social** - Кнопки мессенджеров
- Telegram: `#229ED9` → `#38A8E0` (gradient shift on hover)
- WhatsApp: `#25D366` → `#2FE070` (gradient shift on hover)
- VK: `#0077FF` → `#2787F5` (gradient shift on hover)
- **Особенность**: используют градиент, который плавно смещается при наведении

#### Размеры

- `btn--large`: padding `1.25rem × 3.5rem`, font-size `1.25rem`
- `btn--medium`: padding `1rem × 2.5rem`, font-size `1.125rem` (по умолчанию)
- `btn--small`: padding `0.75rem × 1.75rem`, font-size `1rem`

#### Анимации

При наведении (`:hover`):
- **Два белых шеврона** на фоне (::before и ::after с opacity: 0.4) смещаются вправо на `8px`
- Кнопка поднимается вверх на `2px` (`transform: translateY(-2px)`)
- Усиливается тень
- **Социальные кнопки**: градиент плавно смещается (`background-position: -100% 0%`), создавая эффект перехода цвета
- Transition: `0.3s cubic-bezier(0.4, 0, 0.2, 1)`

**Важно**: Шевроны создаются через CSS псевдоэлементы, поэтому не нужно добавлять их в HTML.

#### Примеры использования

```html
<!-- Основная CTA (красная с шевронами) -->
<a href="#" class="btn btn--primary btn--large">
  Записаться на курс
</a>

<!-- Альтернативная CTA (желтая с шевронами) -->
<button class="btn btn--secondary">
  Оставить заявку
</button>

<!-- Социальная кнопка (без шевронов) -->
<a href="https://t.me/..." class="btn btn--telegram">
  <svg class="btn__icon">...</svg>
  Связаться в TG
</a>

<!-- Группа кнопок -->
<div class="btn-group btn-group--center">
  <a href="#" class="btn btn--primary">Записаться</a>
  <a href="#" class="btn btn--outline">Подробнее</a>
</div>
```

## CSS переменные

Все значения для кнопок определены в `base.css`:

```css
:root {
  /* Кнопки */
  --btn-primary-bg: #f2404f;
  --btn-primary-hover: #ff6670;
  --btn-secondary-bg: #f2d502;
  --btn-secondary-hover: #ffd700;
  --btn-padding-y: 1rem;
  --btn-padding-x: 2.5rem;
  --btn-font-size: 1.125rem;
  --btn-border-radius: var(--radius-action);
  --btn-transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
```

## Тестирование

Откройте `ui-kit.html` в браузере для просмотра всех вариантов кнопок и их состояний.

## Принципы использования

1. **Консистентность**: используйте классы из UI-kit вместо создания собственных стилей
2. **Доступность**: все кнопки поддерживают `:focus-visible` для клавиатурной навигации
3. **Адаптивность**: размеры используют CSS переменные с `clamp()` для fluid typography
4. **Производительность**: анимации используют `transform` и `opacity` для GPU-ускорения
